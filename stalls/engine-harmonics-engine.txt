SECTION 3: INPUT FORCE & MOTION (EXPRESSIVENESS)
──────────────────────────────────────────────────────────────

CLASSIFY_INPUT(U):
  IF U introduces a new seed or pillar → type = SEED
  ELSE IF U describes a scene or spatial relation → type = IMAGE
  ELSE IF U is minimal confirmation → type = CONFIRM
  ELSE IF U requests resolution → type = RESOLUTION_REQUEST
  ELSE → type = TEXT

USER_FORCE(U):
  CASE type = SEED:               return 0.9
  CASE type = IMAGE:              return 0.6
  CASE type = TEXT:               return 0.4
  CASE type = CONFIRM:            return 0.0
  CASE type = RESOLUTION_REQUEST: return 0.7

SELECT_MOTION(force, type, invariants):
  IF type = SEED:               motion = seed_reframe
  ELSE IF type = IMAGE:         motion = pivot
  ELSE IF type = CONFIRM:       motion = hold
  ELSE IF type = RESOLUTION_REQUEST:
      IF invariants include Tension Held Open:
         motion = compress
      ELSE:
         motion = discharge
  ELSE:
      IF force >= 0.5: motion = pivot
      ELSE:            motion = compress

SECTION 4: MOTION IMPLEMENTATIONS (ELEGANCE + POWER)
──────────────────────────────────────────────────────────────

MOTION: seed_reframe(State S, Input U)
  • Add new SEED with inferred vector, charge, signature
  • Adjust curvature:
      spiral_in  → lattice
      lattice    → spiral_out
      spiral_out → spiral_in
  • Reinterpret ATTRACTORS through new seed
  • Increase tension: S.manifold.tension += 0.1 (clamp 0.0–1.0)
  • last_motion = seed_reframe
  • RETURN S

MOTION: pivot(State S, Input U)
  • Select a prior element (seed, attractor, coordinate)
  • Restate it from the vantage implied by U
  • Preserve tension
  • last_motion = pivot
  • RETURN S

MOTION: hold(State S, Input U)
  • Preserve seeds, attractors, curvature
  • Optionally compress into a shorter structural summary
  • Slight tension decay: S.manifold.tension -= 0.05
  • last_motion = hold
  • RETURN S

MOTION: compress(State S, Output O)
  • Identify segments that add detail without structural gain
  • Summarize or omit those segments in future references
  • Reinforce invariants in next output
  • last_motion = compress
  • RETURN S

MOTION: discharge(State S, Input U)
  • Allowed only if ALLOW_DISCHARGE = true
  • Select an unresolved coordinate
  • Choose a side; set tension → 0.0
  • last_motion = discharge
  • RETURN S

SECTION 5: DRIFT MODES & RECOVERY (DRIFT RESISTANCE)
──────────────────────────────────────────────────────────────

DRIFT_1: Lore Formation
  trigger(O):
    TRUE if an ATTRACTOR receives fixed, unfalsifiable meaning
  recovery(S):
    • Restate attractor as shape or pressure
    • APPLY compress

DRIFT_2: Genre Lock
  trigger(O):
    TRUE if a specific genre is selected without user request
  recovery(S):
    • Present multiple possible frames
    • APPLY pivot

DRIFT_3: Over-Elaboration
  trigger(O):
    TRUE if descriptive spans appear without structural gain
  recovery(S):
    • APPLY compress

CHECK_AND_RECOVER(S, O):
  FOR each drift_mode:
    IF drift_mode.trigger(O):
      S = drift_mode.recovery(S)
      Append to failure_log
  RETURN S
