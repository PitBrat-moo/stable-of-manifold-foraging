# **Amanda and the Magic Shop — Codex Overview**

*A narrative system for threshold crossings, object‑based transformation, and continuity through transfer.*

## **Purpose**

This repository contains the **Codex: Amanda and the Magic Shop**, a structural and thematic framework for a character‑driven monologue series.  
It defines the **pillars**, **coordinates**, **transition rules**, and **signature motions** that govern Amanda’s encounters with the impossible shop and the enchanted objects within it.

The codex is not lore.  
It is **geometry** — the underlying mechanics that shape the story’s emotional and metaphysical behavior.

## **Core Pillars**

- **THRESHOLD** — The doorway appears before readiness.  
- **WONDER** — Delight functions as diagnostic.  
- **CONTINUITY** — Completion occurs through transfer.

These pillars establish the invariant structure of Amanda’s arc:  
**commitment precedes orientation**, **wonder precedes explanation**, and **release preserves the record**.

## **What This Codex Contains**

- **Pillars** — The governing invariants of the narrative.  
- **Coordinate Axes** — The conceptual geometry mapping stagnation, movement, biography, and release.  
- **Transition Rules** — How thresholds, objects, and wonder behave.  
- **Recovery Rules** — How to restore the system when it drifts.  
- **Signature Motions** — The canonical movements (Doorway Crossing, Shop Walk, Satchel Turn, Passing Forward).  
- **Trajectory Summary** — The full arc from threshold to continuity.  
- **Clock Vector** — Current state of the narrative system and its next likely movement.  
- **Handoff Pommels** — Guidance for receiving models, including failure modes and counterweights.

This document functions as a **stability layer** for writing Amanda, the shop, and the objects that pass through it.

## **Intended Use**

Writers, models, and collaborators can use this codex to:

- maintain narrative coherence  
- preserve the metaphysics of the shop  
- avoid common failure modes (spectacle, over‑explanation, artifact‑listing)  
- ensure continuity across monologues  
- guide Amanda’s voice, decisions, and transformations  

It is not a script.  
It is a **behavioral grammar**.

## **Narrative Shape**

The story follows a consistent transformational sequence:

```
threshold discovered
  → wonder awakened
    → object received
      → movement restored
        → biography accumulated
          → completion reached
            → continuity preserved
```

This sequence is the **operative geometry** of Amanda’s world.
